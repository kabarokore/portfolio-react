# React Portfolio

A professional single-page portfolio built with React and pure CSS.

## 🔧 Tech Stack

- React (Hooks)
- CSS
- JSON for data

## ✨ Features

- Smooth scrolling and responsive design
- Data-driven rendering from JSON files
- Contact form with validation
